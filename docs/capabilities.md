# Co-Worker 产品能力说明书

> **让你像用 Word 一样用 AI** —— 不用学命令，不用记 prompt，点按钮就行。

---

## 一、Co-Worker 是什么

Co-Worker 是一个**桌面办公工具**。它把 AI 包装成了你熟悉的"技能按钮"：

- 打开 Co-Worker → 看到一排技能卡片
- 点"写周报" → 填几个关键信息 → AI 帮你生成
- 觉得不满意？点👎，AI 记住你的偏好，下次更好

**不需要懂技术。不需要写 prompt。不需要命令行。**

---

## 二、核心能力

### 🏠 技能面板
打开就是技能卡片网格，6 个分类一目了然。你只需要知道"我想做什么事"，不用纠结"AI 能做什么"。

### 📝 引导表单
点技能后不是空白输入框，而是**填空式表单**。每个技能都有专属的引导问题，你填关键信息，AI 处理剩下的。

### 🎨 设计系统
生成 PPT 或文档时，你可以在 **15 套专业设计系统**中选择——想要 Apple 极简风？Stripe 科技紫？Airbnb 温暖橙？一键切换，输出自带品牌级视觉品质。

### ⚡ 实时预览
AI 执行时你能看到**流式进度**——正在分析输入 → 正在生成文档 → 正在排版，每步清清楚楚。

### 👍 越用越好
每次用完后评分 👍/👎，AI 会记住你的偏好。用了 10 次后，AI 自动优化技能——你的 Co-Worker 会越来越懂你。

## 三、技能详单

### 📅 写周报
**一句话：** 填本周工作，生成格式规范的周报。

| 你做的事 | AI 做的事 |
|----------|-----------|
| 填"本周完成的工作" | 分析内容，提取要点 |
| 填"亮点"（选填） | 结构化排版 |
| 选输出风格 | 生成 DOCX 周报 |

**适用场景：** 周五下午不想动脑子的时候。

---

### 📝 写文档
**一句话：** 给大纲，出全文。

| 你做的事 | AI 做的事 |
|----------|-----------|
| 填文档标题 | 按结构展开内容 |
| 列大纲（如：背景→方案→实施） | 补充技术细节 |
| 选风格（正式/简明/通俗） | 生成完整文档 |
| 可选：上传参考文件 | |

**适用场景：** 方案文档、技术说明、会议纪要。

---

### 📊 做PPT
**一句话：** 说你要什么，AI 排版。

| 你做的事 | AI 做的事 |
|----------|-----------|
| 填 PPT 标题 | 逐页生成内容 |
| 描述每页内容 | 应用设计系统风格 |
| 选风格（商务/科技/简约/创意） | 自动排版 |
| 可选：上传参考素材 | 输出 PPTX 文件 |

**✨ 亮点：** 15 套专业设计系统可选，生成的 PPT 自带杂志级排版，不是"白底黑字"。

---

### 📜 写专利交底书（招牌技能）
**一句话：** 从想法到完整交底书，全流程自动化。

**三种输入方式：**
| 模式 | 你做的事 | AI 做的事 |
|------|----------|-----------|
| 💡 天马行空 | 描述你的技术想法 | 头脑风暴→列出3-5个方向→快速查新 |
| 📁 给文档 | 上传技术文档 | 提取技术要点→识别可专利特征 |
| 💻 给代码 | 选项目目录 | 扫描代码结构→识别核心算法 |

**8 步标准流程：** 边界确认 → 技术分析 → 专利点挖掘 → 融合为方法+系统专利 → 国知局查新 → 摘要预览 → 生成全文交底书 → 自检红线

**输出三份文档：** 检索报告 + 技术交底书（九章）+ 调研分析

---

### 🔍 审文档
**一句话：** 上传文档，逐段分析，给出修改建议。

| 你做的事 | AI 做的事 |
|----------|-----------|
| 上传待审文档 | 逐段分析 |
| 选评审维度（逻辑/技术/语言/格式） | 标注问题+严重程度 |
| | 给出修改建议 |
| | 输出评审意见 DOCX |

**严重程度标注：** 🔴 P0 严重 → 🟡 P1 重要 → 🟢 P2 建议

---

### 📧 写邮件
**一句话：** 给要点，出邮件。

| 你做的事 | AI 做的事 |
|----------|-----------|
| 填收件人 | 自动选择称呼 |
| 填主题 | 按语气展开正文 |
| 列要点 | 附行动号召 |
| 选语气（正式/半正式/随意） | 输出正式格式邮件 |

## 四、设计系统库

Co-Worker 内置 15 套专业设计系统（源自 Open Design，Apache-2.0），让 AI 输出自带品牌品质：

| 设计系统 | 风格 | 适合 |
|----------|------|------|
| **Apple** | 极致简约，精密间距 | 产品介绍、技术方案 |
| **Stripe** | 科技紫色的精致金融感 | 商业计划、融资PPT |
| **Notion** | 温暖极简，纸张质感 | 内部文档、知识库 |
| **Airbnb** | 珊瑚粉点缀，摄影驱动 | 品牌提案、创意展示 |
| **Vercel** | 黑白精准，工程师审美 | 技术文档、API说明 |
| **GitHub** | 功能密度，代码感 | 开发者文档 |
| **Cursor** | 温暖深色，AI 原生感 | 技术博客 |
| **Figma** | 设计工具感，多彩 | 设计说明 |
| **Minimal** | 基础极简 | 通用文档 |
| **Corporate** | 企业蓝，稳重 | 正式报告 |
| **Editorial** | 杂志排版 | 长文、白皮书 |
| **Bold** | 大胆对比 | 演讲PPT |
| **Brutalism** | 粗野主义 | 创意表达 |
| **Material** | Google 设计语言 | App 原型说明 |
| **Modern** | 现代清新 | 通用演示 |

## 五、双模交互

| 模式 | 适合谁 | 怎么用 |
|------|--------|--------|
| **普通模式** 🏠 | 办公室白领、产品经理、专利工程师 | 技能卡片→点按钮→填表单→等结果 |
| **高级模式** ⚡ | 开发者、架构师 | 完整对话窗口，自由提问 |

一键切换，状态不丢失。

## 六、与 OpenAI/ChatGPT 的区别

| 对比维度 | ChatGPT | Co-Worker |
|----------|---------|-----------|
| 入口 | 空白输入框 | **场景按钮 + 引导表单** |
| 你问自己 | "我能让 AI 做什么？" | **"我要做这件事"** |
| 输出质量 | 依赖你的 prompt 水平 | **技能模板保证下限** |
| 学习成本 | 需要学写 prompt | **零学习成本，30秒搞定第一个任务** |
| 技能进化 | 无 | **用越多越好，自动优化** |
