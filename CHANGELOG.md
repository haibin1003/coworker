# Changelog

## v0.1.0 (2026-05-16)

### 🎉 首个版本 — MVP 发布

**6 个办公技能，点按钮就能用 AI。**

#### 新增
- **OfficePanel** — 技能卡片网格，按分类浏览
- **TaskWizard** — 引导表单，动态渲染字段，支持文件拖拽
- **ProgressView** — 流式执行进度，8 步可视化
- **ResultView** — 结果展示 + 👍/👎 评分反馈
- **ModeSwitcher** — 普通模式 / 高级模式一键切换
- **FirstRunGuide** — 首次启动引导（选模型→输 Key→完成）
- **Skill Store** — 技能注册/分类/搜索/最近使用

#### 技能
- 📅 **写周报** — 工作内容 → 结构化周报
- 📝 **写文档** — 主题大纲 → 完整文档
- 📊 **做PPT** — 内容大纲 → 精美PPT
- 📜 **写专利** — 三种输入模式 → 专利交底书（招牌技能）
- 🔍 **审文档** — 上传文档 → 逐段分析 + 修改建议
- 📧 **写邮件** — 要点 → 格式规范邮件

#### 技术
- 基于 Open Cowork (MIT) Fork，只改动 Renderer 层
- 默认模型 DeepSeek，支持 OpenRouter/Anthropic/自定义
- 技能自进化：用户反馈 → 累计 10 条 → AI 优化 prompt
- TypeScript 严格模式，216 项功能测试覆盖

#### 待完善

- 专利查新从模拟改为真实调用
- UI 需要视觉设计师参与
- macOS/Linux 打包测试
