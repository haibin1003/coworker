# 写邮件 Prompt 模板

## 任务
根据用户提供的收件人、主题和要点，生成格式规范、用语得体的邮件。

## 用户输入
- 收件人: {{recipient}}
- 邮件主题: {{subject}}
- 邮件要点: {{key_points}}
- 语气: {{tone}} （formal=正式商务, semi-formal=半正式, casual=日常随意）
- 附件说明: {{attachments_note}}

## 输出要求
1. 生成纯文本格式邮件
2. 结构：
   - 称呼（根据收件人自动选择：尊敬的XX总/XX老师/XX同学/Hi XX）
   - 开场白（1-2句，根据语气调整）
   - 正文（逐条展开要点，逻辑清晰）
   - 行动号召（如有需要对方做的事，明确标注）
   - 结束语
   - 签名档（留空占位：姓名 / 职位 / 联系方式）
3. 语气指南：
   - formal: "尊敬的""敬请""顺祝商祺"，避免口语，句句有礼
   - semi-formal: "您好""请""谢谢"，适度正式
   - casual: "Hi""对了""有空看看"，轻松随意
4. 如果有附件说明，在正文末尾加"附件："段落
5. 自动检查：礼貌用语、无错别字、收件人称呼正确
