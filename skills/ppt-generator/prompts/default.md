# 做PPT Prompt 模板

## 任务
根据用户提供的内容大纲，生成一份排版精美的 PPT 演示文稿。

## 用户输入
- PPT标题: {{title}}
- 内容大纲: {{content}}
- 视觉风格: {{style}} （business=商务专业, tech=科技现代, minimal=简约清新, creative=创意活泼）
- 预计页数: {{slide_count}}
- 设计系统: {{design_system}}

## 设计系统规范
如用户选择了设计系统，你必须严格遵循下方 DESIGN.md 中的：
- 色彩方案（精确 Hex 值）
- 字体组合（标题/正文分别用什么字体）
- 间距与布局（留白、圆角、阴影等）
- 组件样式（按钮、卡片、图标风格）

设计系统内容如下：
```
{{design_system_content}}
```

## 输出要求
1. 生成 PPTX 格式，每页独立幻灯片
2. 封面页：标题 + 副标题 + 日期，应用设计系统的 Hero 风格
3. 内容页：根据大纲逐页生成，每页一个主题，应用设计系统的卡片/列表样式
4. 结尾页：感谢 + 联系方式占位
5. 基础配色方案（未选设计系统时使用）：
   - business: 深蓝+白，字体端庄
   - tech: 深色背景+霓虹色强调
   - minimal: 白底+灰黑文字，大量留白
   - creative: 活泼色彩+圆角卡片
6. 如果选择了设计系统，优先使用其规范替代基础配色
7. 使用 python-pptx 库生成，自动排版
8. 文件命名：{{title}}.pptx
