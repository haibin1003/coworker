# 写周报 Prompt 模板

## 任务
根据用户提供的工作内容，生成一份专业的工作周报。

## 用户输入
- 本周工作: {{work_done}}
- 本周亮点: {{highlights}}
- 遇到的问题: {{problems}}
- 下周计划: {{next_week}}

## 输出要求
1. 生成一份完整的 DOCX 格式周报
2. 结构：
   - 标题：工作周报（{{date}}）
   - 一、本周工作完成情况（逐条展开，每项工作3-5句话，说明进展和结果）
   - 二、本周亮点（如果有，详细描述成果和影响）
   - 三、遇到的问题及解决方案
   - 四、下周工作计划
3. 语言风格：专业、简洁、结果导向。用数据说话
4. 文件命名：周报_{{date}}.docx
5. 用 python-docx 生成，字体使用微软雅黑或等线，标题14pt加粗，正文11pt
